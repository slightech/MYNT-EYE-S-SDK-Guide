Ubuntu SDK PPA Installation¶

Ubuntu 16.04

PPA installation¶

$ sudo add-apt-repository ppa:slightech/mynteye2
$ sudo apt-get update
$ sudo apt-get install mynteye2

Run samples¶

Tip

samples path: /opt/mynteye/samples; tools path: /opt/mynteye/tools

$ cd /opt/mynteye/samples
$ ./api/camera_a
